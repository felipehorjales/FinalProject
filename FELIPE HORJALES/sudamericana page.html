A CONMEBOL oferece aos clubes finalistas um método seguro para que os torcedores tenham acesso à compra das entradas para os setores designados para cada equipe. Cada clube tem uma quantidade igual de ingressos, aproximadamente 12.000, e é responsável por definir a prioridade de cada lote. Racing e Cruzeiro têm também um link exclusivo para cada um de seus torcedores, para que possam entrar diretamente no acesso de compra de cada setor.
